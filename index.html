Theory: What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using tags like <h1> for headings and <p> for paragraphs.

DOCTYPE: Declares the HTML version to ensure proper rendering. In HTML5, it's simplified as <!DOCTYPE html>.

World Wide Web

The World Wide Web (WWW) was invented by Tim Berners-Lee. It uses three main technologies: HTML (for web page structure), URL (for addressing web resources), and HTTP (for data transfer).

Unified Resource Locator (URL)

A URL is the address of a resource on the web. Its syntax consists of the protocol (e.g., https://), domain name (e.g., www.example.com), and optional paths or query strings.

Web Server

A web server stores, processes, and delivers web pages to clients. It handles requests from browsers using HTTP or HTTPS.

Web Browser

A web browser is a client software used to access web pages. It processes HTML, CSS, and JavaScript to display websites and interacts with web servers to fetch content.

Other facts are for example that the first website was created in 1991 by Tim Berners-Lee and it was about the World Wide Web project itself.

Combiantors are used to select elements with a specific attribute value. For example, p[title="hello"] selects all paragraph elements with title="hello"
or p[title~="hello"] selects all paragraph elements with title containing the word hello. All the known combinators are: descendant, child, adjacent sibling, general sibling. the description of the combinators is:
descendant - Selects all elements that are descendants of a specified element
child - Selects all direct children of a specified element
adjacent sibling - Selects all elements that are the adjacent siblings of a specified element
general sibling - Selects all elements that are siblings of a specified element

Theory: What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML.

Cascading: Refers to the order in which styles are applied. Styles can be inherited from parent elements or overwritten by more specific rules.

Box Model

The CSS box model describes the rectangular boxes that are generated for elements in the document tree. It consists of content, padding, border, and margin areas.

Flexbox

Flexbox is a layout model that allows elements to align and distribute space within a container. It's ideal for one-dimensional layouts like navigation bars or form elements.

Grid

CSS Grid Layout is a two-dimensional layout system that allows elements to be placed in rows and columns. It's ideal for complex layouts like web pages or applications.

The display property specifies the display behavior of an element. It can be set to block, inline, inline-block, flex, grid, inline-flex, inline-grid, table, table-row, table-cell, none, inherit, or initial.

The position property specifies the type of positioning method used for an element. It can be set to static, relative, absolute, fixed, sticky, inherit, or initial.

Pseudo classes are used to define a special state of an element. For example, :hover is used to apply styles when a user hovers over an element. Other pseudo classes include :active, :focus, :first-child, :last-child, :nth-child(), :not(), :checked, :disabled, :empty, :target, :visited, :before, and :after.

All the pseudo classes and their description are:
:hover - Selects an element when you mouse over it
:active - Selects the active link
:focus - Selects the element that has focus
:first-child - Selects the first child element of its parent
:last-child - Selects the last child element of its parent

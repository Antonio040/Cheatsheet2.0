Theory: What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using tags like <h1> for headings and <p> for paragraphs.

DOCTYPE: Declares the HTML version to ensure proper rendering. In HTML5, it's simplified as <!DOCTYPE html>.

World Wide Web

The World Wide Web (WWW) was invented by Tim Berners-Lee. It uses three main technologies: HTML (for web page structure), URL (for addressing web resources), and HTTP (for data transfer).

Unified Resource Locator (URL)

A URL is the address of a resource on the web. Its syntax consists of the protocol (e.g., https://), domain name (e.g., www.example.com), and optional paths or query strings.

Web Server

A web server stores, processes, and delivers web pages to clients. It handles requests from browsers using HTTP or HTTPS.

Web Browser

A web browser is a client software used to access web pages. It processes HTML, CSS, and JavaScript to display websites and interacts with web servers to fetch content.

CSS Cheat Sheet

  • color - Sets text color
  • background-color - Sets background color
  • font-size - Sets font size
  • margin - Sets outside space of elements
  • padding - Sets inside space of elements
  • border - Sets border around elements
  • text-align - Aligns text
  • display - Sets how elements are displayed
  • position - Sets element position
  • float - Floats elements
  • flexbox - Aligns elements in a flexible container
  • grid - Aligns elements in a grid container
  • animation - Animates elements
  • transition - Adds transition effects
  • transform - Transforms elements in a way that they are displayed like this rotate(45deg)
  • box-shadow - Adds shadow to elements
  • text-shadow - Adds shadow to text
  • filter - Adds visual effects like blur or grayscale
  • opacity - Sets element transparency
  • z-index - Sets element stacking order
  • overflow - Sets how content that overflows an element is handled
  • cursor - Sets mouse cursor type
  • user-select - Sets if the user can select text
  • pointer-events - Sets if an element can be clicked
